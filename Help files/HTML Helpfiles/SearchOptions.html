Configurations


Startscreen with settings for the scan
  1. Path to parent folder where subfolders contain SQLite databases.
  2. Here all the subfolders get listed, with the option to select one, several or all. The radiobuttons gives you the option to select all databases (automatically checks the checkbox "Select all folders") or select specific ones (at least one folder must be selected).
  3. Include types in your search. See search types. The radiobutton "All search types" automatically selects all.
  4. Include a timeline in the search, by setting a start/end date and time.
  5. Include additional search strings.
    This could be locations, name on images, e-mail content, etc.
    Seperate each search string by newline (it is not possible to search for complete sentences).
  6. Search buttons:
    • 'Start search': Starts the search (Path must be set, and at least one folder and search type selected).
    • 'Cancel search': Cancels the started/ongoing search.
    • 'Reset Parameters': Resets the settings back to default.
  7. Listbox displaying information and error messages.
