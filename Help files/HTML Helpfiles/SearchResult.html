The resultscreen when scan is finished

  1. Treeview:
    Displays the parent folder, subfolders, databases contained within subfolders and the tables belonging to the given database.
  2. Possibility to search within the results, see searching within results. Labels displays the:
    • Number of databases that were searched
    • Number of hits/results based on search criteria(s)
  3. Datagridview: The results are displayed here.
    The number of columns, content and description may vary depending on search and results.
