Startpage

This was a development project in the course IMT4641 Computational Forensic at Gjøvik University College. The basis for my project was to develop a program using machine learning that could be used when investigating Android devices. The focus of the project is searching through the SQLite databases for content or evidence that may be of interest during an investigation.

Features
  • Search one or more SQLite databases at the same time for evidence.
  • Search for evidence based on different types.
  • Search for evidence based on date and time (currently not implemented).
  • Search for evidence based on search strings.
  • Or search for evidence based on all above.
